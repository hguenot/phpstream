Detailed Description

A filter operator is used during the Stream Processor to check if element could continue the process or not

Constructor & Destructor Documentation

phpstream\operators\FilterOperator::__construct ( callable|UnaryFunction|FilterOperator  $fn)

Construct a new operator with the filter function.

If parameter is a callable, the function should take one argument (value in the collection) and returns a boolean (function(mixed $value): bool) If parameter is an UnaryFunction, the implemented UnaryFunction::apply should return a boolean value.

The value will be kept only if the return value of the filter method is considered as true
