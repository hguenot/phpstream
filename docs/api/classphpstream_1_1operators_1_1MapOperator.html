phpstream\operators\MapOperator::__construct (   $fn)

Creates a new MapOperator using the given function.

If parameter is a callable, the function should take one argument (value in the collection) and returns another value (function(mixed $value): mixed). If parameter is an UnaryFunction, the implemented UnaryFunction::apply should return the mapped value. If parameter is a string, the process will use the corresponding property (or method if property does not exist) for getting the mapped value.

The returned value will replace the current value for the rest of process.
